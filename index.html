Counting Paragraphs

This page uses a simple script to count and report the number of paragraph tags it contains.

The script is run when the page is loaded.

It might look like the values reported below are part of the original source, but in fact they are calculated!

Method 1 (nodeValue): There are n/a paragraph tags. This directly changes the text of a textNode within a span tag.

Method 2 (innerText): There are n/a paragraph tags. This replaces existing text within a span tag. Note that any HTML we provide (e.g. an em tag) is not interpreted.

Method 3 (innerHTML): There are n/a paragraph tags. This replaces existing text within a span tag. Note that this method interprets HTML we provide, hence the link!
